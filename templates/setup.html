To get started, you need a GitHub Personal Access Token.

How to get your GitHub token:

  1. Go to GitHub Fine-Grained Token Generator.
  2. Select the necessary scopes (e.g., `repo` for repository access).
  3. Set an expiration for the token, if desired (for security purposes, consider setting an expiration date).
  4. Click Generate token, then copy the token.
  5. Paste it in the box below.
